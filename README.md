# Rust Calculator

A fast mathematical expression calculator, written in Rust.

## Features

- Basic operations: `+ - * / ^ ( )`
- Math functions: `sin(), cos(), sqrt(), print()`
- Variables: `let x = 5`
- Constants: `pi`, `e`

## Quick Start

```bash
# Clone and run
git clone https://github.com/dxavvv/rust-calc
cd rust-calc
cargo run
```
